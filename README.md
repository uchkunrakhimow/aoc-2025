# Advent of Code 2025

TypeScript solutions with Bun.

## Structure
```
dayXX/
  ├── index.ts       # Solution
  ├── index.test.ts  # Tests
  ├── input.txt      # Puzzle input
  └── sample.txt     # Example
```

## Usage
```bash
bun run dayXX     # Run solution
bun test          # Run all tests
```
